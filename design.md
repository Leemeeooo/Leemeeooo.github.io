# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) and warm ivory (#f8f6f0)
- **Accent**: Muted coral (#d4a574) and sage green (#8b9a8b)
- **Supporting**: Soft lavender (#c8b5d1) and warm gray (#9b9b9b)
- **Contrast**: High contrast text (4.5:1 ratio) for accessibility

### Typography
- **Display Font**: "Canela" serif for headings - bold, editorial presence
- **Body Font**: "Suisse Int'l" sans-serif for content - clean, readable
- **Accent Font**: "Tiempos Headline" for special emphasis
- **Hierarchy**: Large display headings, medium subheadings, small body text

### Visual Language
- **Editorial Inspiration**: Kinfolk, Wired, and The Gentlewoman aesthetics
- **Layout**: Grid-based, asymmetric compositions with generous white space
- **Imagery**: High-contrast, editorial-style photography and abstract visuals
- **Textures**: Subtle paper grain and organic shapes for depth

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and page transitions
- **ECharts.js**: Interactive radar chart for skills visualization
- **Typed.js**: Typewriter effect for hero section
- **Splitting.js**: Text animation effects
- **p5.js**: Creative coding background elements

### Animation Effects
- **Scroll Animations**: Staggered reveal of content sections
- **Hover Effects**: 3D tilt transforms on cards, color morphing on buttons
- **Text Effects**: Typewriter animation for hero title, character-by-character reveals
- **Background**: Subtle particle system with organic movement

### Styling Approach
- **Grid System**: CSS Grid with responsive breakpoints
- **Transitions**: 300ms ease-out for hover states
- **Shadows**: Soft, layered shadows for depth
- **Borders**: Minimal, using color blocks instead of heavy lines

### Header Effect
- **Background**: Animated gradient mesh with subtle color shifts
- **Typography**: Large display text with typewriter animation
- **Composition**: Asymmetric layout with floating elements

### Interactive Elements
- **Buttons**: Rounded corners with gradient backgrounds and hover morphing
- **Cards**: Lift effect with shadow expansion on hover
- **Forms**: Floating labels with focus animations
- **Navigation**: Smooth underline transitions and active states
